# Metrobots Rookie Robot 2026 Codebase

This is the code of the rookie robot for First Robotics Competition team 3324, The Metrobots.

# Subsystems

## Drivetrain

We are using two MAXSwerve modules and two caster wheels. The code for the drivetrain is based on the [official Java template](https://github.com/REVrobotics/MAXSwerve-Java-Template) provided by Rev Robotics. Currently, the PID values aren't tuned (they're the default from the template). It also hasn't been tested yet.
